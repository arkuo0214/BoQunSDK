2022.11.01

v2.1.2 —> v2.1.9 Change log.

Discarded API, which will be deleted in a later version. Please replace it with a new API as soon as possible

  • Added BoQunTreadmill.init(OnTreadmillListener listener) to replace BoQunTreadmill.init() and BoQunTreadmill.setOnTreadmillListener(OnTreadmillListener)

  • Deprecated BoQunTreadmill.init()

  • Deprecated BoQunTreadmill.setOnTreadmillListener(OnTreadmillListener)

  • Added BoQunRower.init(OnRowerListener listener) to replace BoQunRower.init() and BoQunRower.setOnRowerListener(OnRowerListenerl)

  • Deprecated BoQunRower.init()

  • Deprecated BoQunRower.setOnRowerListener(OnRowerListenerl)

  • Added BoQunBike.init(OnBikeListener listener) to replace BoQunBike.init() and BoQunBike.setBikeListener(OnBikeListener l)

  • Added BoQunBike.setSportData(time, speed, distance, calories, rpm, pulse, watt)

  • Added BoQunBike.setSportParams(code,value)

  • Deprecated BoQunBike.init()

  • Deprecated BoQunBike.setBikeListener(OnBikeListener l)
