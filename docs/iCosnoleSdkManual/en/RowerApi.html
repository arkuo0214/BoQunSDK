Rower

Rower API

BoQunRower

Parameter Description
BoQunRower.TYPE_SPORT_STATUS Sports status, callback onResponseSportStatus
BoQunRower.TYPE_LOAD_LEVEL Resistance level, callback onResponseLoadLevel
Function Description
Instantiates the rowing machine and returns a Boolean value indicating whether the serial port was successfully initialized.
BoQunRower.queryConfigInfo() Query the rowing machine configuration information resistance, fan status, etc., callback onResponseConfigInfo, read failure callback onError
BoQunRower.queryMachineInfo(int type) Query rowing machine information
BoQunRower.start() Start exercising, call back onResponseSportStatus
BoQunRower.pause() Pause the sport and call back onResponseSportStatus
BoQunRower.stop() Stop exercising and call back onResponseSportStatus
BoQunRower.setLoadLevel(int level) Control the resistance level of the rowing machine, callback onResponseLoadLevel, the larger the level value, the greater the resistance, the level has a range limit, the level range Parameter reads RowerConfigInfo
BoQunRower.setFanLevel(int level) Control the fan wind speed level, callback onResponseFanLevel, the wind speed level supports 0-3 levels, if the machine does not have a fan, it does not support adjustment, get whether the machine has a fan status RowerConfigInfo
BoQunRower.setWorkMode(@RowerWorkMode int mode) Set the rowing machine lower computer working mode
BoQunRower.setFTMSEnabled(boolean enable) Enable or disable FMTS. It must be used with a lower control panel with FMTS function. Callback RowerStatusCallbacks.FTMSStateCallback
BoQunRower.setSportData(long time, float speed, float distance, int calories, int spm, int pulse, int watt) Pass motion data to the lower control
BoQunRower.registerStateListener(RowerStatusCallback callback) Registering a motion status listener
BoQunRower.unregisterStateListener(RowerStatusCallback callback) Unregister the motion status listener
BoQunRower.unregisterAllStateListener() Unregister all motion state listeners
BoQunRower.setOnRowerListener(OnRowerListener onRowerListener) Rowing Machine Monitor
BoQunRower.setSerialPortEnabled(boolean enable) Enable and disable the serial port. The default is enabled. This method is provided to facilitate handling special situations. For example, multiple APPs reading the serial port at the same time will cause mutual interference. This method can avoid

tips: It will not really close the serial port, but it will definitely not read the serial port data
BoQunRower.destroy() Destroy the instance and release resources

RowerStatusCallback

Function Description
void onRowerSportStateChange(int status) Changes in motion status

RowerStatusCallbacks

This class mainly implements status callback. It currently supports monitoring FTMS status. Other functions will be implemented later if needed, or you can inherit the RowerStatusCallback interface to implement custom status callback.

Currently implements FMTS switch status callback

BoQunRower.registerStateListener(new RowerStatusCallbacks.FTMSStateCallback() {
            @Override
            public void onFTMSStateChange(boolean enable) {
                 //Enable and disable status
            }
        });

Remember to unregister

BoQunRower.unregisterStateListener(xxxx);

//Or, look at your personal application scenarios and needs
BoQunRower.unregisterAllStateListener();

RowerWorkMode

Parameter Description
RowerWorkMode .NORMAL This state should be set when the TFT screen is on and enters the working state
RowerWorkMode .SLEEP This state should be set when the TFT screen is off and enters sleep mode
RowerWorkMode .SHUT_DOWN This state should be set when TFT is turned off to release resources

OnRowerListener

Function Description
void onResponseConfigInfo(RowerConfigInfo info) Reply to the control configuration information
void onResponseSportStatus(@RowerSportState int status) Resume exercise status
void onResponseSportData(RowerSportData data) Reply to sports data
void onResponsePullAndPut(@IntRange(from = 0, to = 100) int progress, boolean isPull) Respond to the pull and release status and progress value
progress: pull and release stroke percentage

isPull = true for pull, isPull = false for release
void onResponseLoadLevel(int load) Resume resistance level
void onResponseFanLevel(int level Reset fan speed level
void onExternalKeyEvent(@RowerKeyCode int keyCode) Pressing the external key on the panel triggers the callback (some UP and DOWN buttons can be long pressed)
void onError(int errorCode) Error callback

RowerSportState

Parameter Description
RowerSportState .STOP Stop state
RowerSportState .START Starting state
RowerSportState .PAUSE Pause state

BoQunRowerFactory

Function Description
BoQunRowerFactory.init() Instantiating the Rowing Machine Factory
BoQunRowerFactory.queryFactoryInfo() Query the rowing machine factory information and call back onResponseFactoryInfo
BoQunRowerFactory.queryLoadMotorStroke(int load) Query the motor stroke and call back onResponseLoadMotorStroke
BoQunRowerFactory.setLoadMotorStroke(int load, int adc) Setting the motor stroke
BoQunRowerFactory.setRepairDirection(@RowerRepairDirection int direction) Repair motor error
direction:

- RowerRepairDirection.UP Repairs the upper side
- RowerRepairDirection.DOWN Repairs the lower side
BoQunRowerFactory.setFactoryInfo(RowerFactoryInfo info) Modify rowing machine factory information
BoQunRowerFactory.setOnRowerFactoryListener(OnRowerFactoryListener listener) Rowing Machine Factory Monitor
BoQunRowerFactory.destroy() Destroy factory pattern instance

OnRowerFactoryListener

Function Description
void onResponseFactoryInfo(RowerFactoryInfo info) Reply factory information, ParameterRowerFactoryInfo
void onResponseLoadMotorStroke(int load, int adc) Resume motor stroke, Parameter: load, adc
void onResponseMotorStatus(int status) Resume motor status, Parameter: status

RowerMotorStatus

When an error occurs, use BoQunRowerFactory.setRepairDirection(@RowerRepairDirection int direction) to fix the error

Parameter Description
RowerMotorStatus.NORMAL normal
RowerMotorStatus.ERROR_UP Error above
RowerMotorStatus.ERROR_DOWN Error below

RowerSportData

Parameter Type Description
getTime() long Exercise time (unit: seconds)
getSpm() int Paddle speed per minute
getStroke() int Number of strokes
getWatt() int Power (unit: W)
getDistance() int Movement distance, (unit: m)
getCalories() int Calories burned during exercise (unit: kcal)
getTimePer500m() long How much time per 500 meters (unit: seconds)
getHeartRate() int Heart rate

RowerConfigInfo

Parameter Type Description

hasFan() boolean With fan
isManualControl() boolean Manual Control
getMinLoadLevel() int Minimum resistance
getMaxLoadLevel() int Maximum resistance
getMinWatt() int Minimum WATT
getMaxWatt() int Maximum WATT

RowerFactoryInfo

Parameter Type Description
getDiameter() int diameter
getMagnet() int magnet
getDistance() int Distance Ratio
getCalories() int Calorie Ratio
getTimePer500m() int How much time is required for each 500 meters
