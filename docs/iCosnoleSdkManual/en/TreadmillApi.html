Treadmill

Treadmill API

BoQunTreadmill

Parameter Description
BoQunTreadmill.TYPE_SPORT_STATUS Sports status, callback onResponseSportStatus
BoQunTreadmill.TYPE_SPEED_AND_INCLINE Speed ​​and ascension, callbacks onResponseSpeedLevel and onResponseInclineLevel
Function Description
BoQunTreadmill.init(OnTreadmillListener l) Initializes the treadmill, returning a boolean indicating whether the serial port was successfully initialized.
BoQunTreadmill.queryConfigInfo() Query the treadmill configuration information, including the maximum and minimum speeds, lifting range, etc., and call back onResponseConfigInfo. If reading fails, call back onError
BoQunTreadmill.queryMachineInfo( int type) Query treadmill information by type
BoQunTreadmill.start() Start exercising, call back onResponseSportStatus
BoQunTreadmill.pause() Pause the sport and call back onResponseSportStatus
BoQunTreadmill.stop() Stop exercising and call back onResponseSportStatus
BoQunTreadmill.coolDown() CoolDown mode, callback onResponseSportStatus and onCoolDownStateChange
BoQunTreadmill.setSpeedLevel( int speed) Set the treadmill speed and call back onResponseSpeedLevel
speed 1 = 0.1 km
BoQunTreadmill.setInclineLevel( int incline) Set the treadmill to rise, and call back onResponseInclineLevel
BoQunTreadmill.setFanLevel(int level) Set the fan wind speed level, callback onResponseFanLevel, the wind speed level supports 0-3 levels, if the machine does not have a fan, it does not support adjustment, get whether the machine has a fan status TreadmillConfigInfo
BoQunTreadmill.setWorkMode(@TreadmillWorkMode int mode) Set the lower computer working mode
BoQunTreadmill.setFTMSEnabled(boolean enable) Enable or disable FMTS. It must be used with a lower control board with FMTS function. Callback TreadmillStatusCallbacks.FTMSStateCallback
BoQunTreadmill.setOnTreadmillListener(OnTreadmillListener l) Set up a treadmill listener (deprecated)
BoQunTreadmill.setSerialPortEnabled(boolean enable) Enable and disable the serial port. The default is enabled. This method is provided to facilitate handling special situations. For example, multiple APPs reading the serial port at the same time will cause mutual interference. This method can avoid

tips: It will not really close the serial port, but it will definitely not read the serial port data
BoQunTreadmill.registerStateListener(TreadmillStatusCallback callback) Register status callback listener
BoQunTreadmill.unregisterStateListener(TreadmillStatusCallback callback) Unregister status callback listener
BoQunTreadmill.unregisterAllStateListener() Unregister all status callback listeners
BoQunTreadmill.destroy() Destroy the instance and release resources

TreadmillWorkMode

Parameter Description
TreadmillWorkMode.NORMAL This state should be set when the TFT screen is on and enters the working state
TreadmillWorkMode.SLEEP This state should be set when the TFT screen is off and enters sleep mode
TreadmillWorkMode.SHUT_DOWN This state should be set when TFT is turned off to release resources

TreadmillStatusCallback

This is an interface class, treadmill state change and motion state change callback

Function Description
void onTreadmillSportStateChange(int state); Motion state change callback
void onTreadmillStateChange(int state,int speed); Treadmill status changes

TreadmillStatusCallbacks

​ This class mainly implements some abstract class monitoring status callbacks. Currently, it supports security key status monitoring, CoolDown mode, FTMS status monitoring, and unmanned motion monitoring. Other functions will be implemented later if needed, or you can inherit the TreadmillStatusCallback interface to implement custom status callbacks.

Currently, FMTS switch status callbacks are implemented

//Security KEY status monitoring
BoQunTreadmill.registerStateListener(new TreadmillStatusCallbacks.SafetyKeyStatusCallback() {
       @Override
       public void onSafetyKeyStatus(boolean normal) {
            //normal==true //Security Key Insertion
            //normal==false //Safety Key removal
       }
});
//CoolDown Mode Monitoring
BoQunTreadmill.registerStateListener(new TreadmillStatusCallbacks.CoolDownStatusCallback() {
       /**
        * status:

        * CoolDownState.START    COOL DOWN mode starts to slow down
        * CoolDownState.ING      During deceleration, the speed value will become smaller and smaller
        * CoolDownState.END      Cool down mode ends
        *
        * speed:Real-time speed
        */
       @Override
       public void onCoolDownStatus(int status, int speed) {

       }
});

//Unmanned motion monitoring
long detectionTime=45;//Detection time of no movement, unit: second

BoQunTreadmill.registerStateListener(
    new TreadmillStatusCallbacks.UnmannedStateDetectionCallback(detectionTime) {
       @Override
       public void onUnmannedStateDetection(boolean isUnmanned, long time) {
           //isUnmanned Whether there is no movement at present. When the time of no movement exceeds the detection time, it will become true.
           //time Start counting the time when no one is exercising
       }
});

//FTMS Enable status monitoring
BoQunTreadmill.registerStateListener(new TreadmillStatusCallbacks.FTMSStateCallback() {
            @Override
            public void onFTMSStateChange(boolean enable) {
                Log.d(TAG, "onFTMSStateChange: " + enable);
            }
        });
        BoQunTreadmill.setFTMSEnabled(true);

Remember to unregister

BoQunTreadmill.unregisterStateListener(xxxx);

//Or, look at your personal application scenarios and needs
BoQunTreadmill.unregisterAllStateListener();

OnTreadmillListener

Function Description
void onResponseConfigInfo(TreadmillConfigInfo info) Reply to the control configuration information
void onResponseSportStatus(@TreadmillSportState int state) Resume exercise status
void onResponseSportData(int heartRate, int speed) Reply to sports data
heartRate: heart rate
speed: real-time speed
void onResponseSpeedLevel(int speed) Resume down control speed level
void onResponseInclineLevel(int incline) Resume control and ascend level
void onResponseFanLevel(int level) Reset fan speed level
void onExternalKeyEvent(@TreadmillKeyCodeint keyCode, int value) Pressing the external keys on the panel (some UP and DOWN buttons can be long pressed) triggers the callback
void onError(int errorCode) Error callback ErrorCode, the treadmill will also call back when an error occurs, but the manufacturer is requested to provide an error code list for the down-control communication error code

TreadmillSportState

Parameter Description
TreadmillSportState.STOP Stop state
TreadmillSportState.START Starting state
TreadmillSportState.PAUSE Pause state
TreadmillSportState.COOL_DOWN In COOL DOWN mode
TreadmillSportState.ERROR An error occurred

BoQunTreadmillFactory

Function Description
BoQunTreadmillFactory.init() Instantiating the Treadmill Factory
BoQunTreadmillFactory.queryFactoryInfo() Query treadmill factory information and call back onResponseFactoryInfo
BoQunTreadmillFactory.setFactoryInfo(int code, int value) Set treadmill factory information
BoQunTreadmillFactory.startAutoCorrection() Start automatic correction, callback onResponseMotorAutoCorrection
BoQunTreadmillFactory.stopAutoCorrection() Stop automatic correction, callback onResponseMotorAutoCorrection
BoQunTreadmillFactory.setOnTreadmillFactoryListener(OnTreadmillFactoryListener listener) Treadmill Factory Listener
BoQunTreadmillFactory.destroy() Destroy the treadmill factory instance

OnTreadmillFactoryListener

Function Description
void onResponseFactoryInfo(TreadmillFactoryInfo info) Query factory information callback
void onResponseMotorAutoCorrection(@AutoCorrectionState int status, int speed, int adc, int inclineVR) Autocorrection callback

TreadmillConfigInfo

Parameter Type Description
getWattGroup() int Get WATT Group
getMinInclineLevel() int Get the minimum incline
getMaxInclineLevel() int Get the maximum incline
getMinSpeedLevel() int Get minimum speed
getMaxSpeedLevel() int Get the maximum speed
getChipSpeedLevel() int Get the speed increment. When accelerating or decelerating, you should use the current getChipSpeedLevel() to add or subtract
hasIncline() boolean Is there incline?
hasFan() boolean Is there a fan?
