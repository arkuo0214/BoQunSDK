跑步机

跑步机API

BoQunTreadmill

参数 说明
BoQunTreadmill.TYPE_SPORT_STATUS 运动状态，回调onResponseSportStatus
BoQunTreadmill.TYPE_SPEED_AND_INCLINE 速度和扬升，回调onResponseSpeedLevel和onResponseInclineLevel
函数 说明
BoQunTreadmill.init(OnTreadmillListener l) 初始化跑步机，返回一个布尔值，表示串行端口是否成功初始化。
BoQunTreadmill.queryConfigInfo() 查询跑步机配置信息，最大最小速度和扬升范围等等，回调onResponseConfigInfo，读取失败回调onError
BoQunTreadmill.queryMachineInfo( int type) 查询跑步机信息，回调onResponseConfigInfo
BoQunTreadmill.start() 开始运动，回调onResponseSportStatus
BoQunTreadmill.pause() 暂停运动，回调onResponseSportStatus
BoQunTreadmill.stop() 停止运动，回调onResponseSportStatus
BoQunTreadmill.coolDown() CoolDown模式，回调onResponseSportStatus和onCoolDownStateChange
BoQunTreadmill.setSpeedLevel( int speed) 设置跑步机速度，回调onResponseSpeedLevel
speed 1= 0.1公里
BoQunTreadmill.setInclineLevel( int incline) 设置跑步机扬升，回调onResponseInclineLevel
BoQunTreadmill.setFanLevel(int level) 设置风扇风力级别，回调onResponseFanLevel，风力级别支持0-3段，机台没有风扇则不支持调整，获取机台是否有风扇状态TreadmillConfigInfo
BoQunTreadmill.setWorkMode(@TreadmillWorkMode int mode) 设置下位机工作模式
BoQunTreadmill.setFTMSEnabled(boolean enable) 启用或关闭FMTS，须搭配带含有FMTS功能的下控板使用，回调TreadmillStatusCallbacks.FTMSStateCallback
BoQunTreadmill.setOnTreadmillListener(OnTreadmillListener l) 设置跑步机监听器（已弃用）
BoQunTreadmill.setSerialPortEnabled(boolean enable) 启用和禁用串口，默认是启用状态，提供此方法是方便处理特殊情况，比如多个APP同时读取串口会造成互相干扰问题，使用此方法可避免

tips：并不会真正关闭串口，但绝对不会读取串口资料
BoQunTreadmill.registerStateListener(TreadmillStatusCallback callback) 注册状态回调监听器
BoQunTreadmill.unregisterStateListener(TreadmillStatusCallback callback) 解除注册状态回调监听器
BoQunTreadmill.unregisterAllStateListener() 解除所有注册状态回调监听器
BoQunTreadmill.destroy() 销毁实例，释放资源

TreadmillWorkMode

参数 说明
TreadmillWorkMode.NORMAL TFT屏幕亮屏时应设置该状态,进入工作状态
TreadmillWorkMode.SLEEP TFT屏幕熄屏时应设置该状态,进入休眠模式
TreadmillWorkMode.SHUT_DOWN TFT关机时应设置该状态,释放资源

TreadmillStatusCallback

这是一个接口类，跑步机状态变化和运动状态变化回调

函数 说明
void onTreadmillSportStateChange(int state); 运动状态变化回调
void onTreadmillStateChange(int state,int speed); 跑步机状态变化

TreadmillStatusCallbacks

​ 这个是类主要实现一些抽象类监听状态回调，目前支持安全Key状态监听,CoolDown模式，FTMS状态监听，无人运动监听，后续有需求再实现其它功能或者你可以继承TreadmillStatusCallback接口实现自定义状态回调

目前实现有FMTS开关状态回调

//安全KEY状态监听
BoQunTreadmill.registerStateListener(new TreadmillStatusCallbacks.SafetyKeyStatusCallback() {
       @Override
       public void onSafetyKeyStatus(boolean normal) {
            //normal==true //安全Key插入
            //normal==false //安全Key拔出
       }
});
//CoolDown模式监听
BoQunTreadmill.registerStateListener(new TreadmillStatusCallbacks.CoolDownStatusCallback() {
       /**
        * status:

        * CoolDownState.START    COOL DOWN模式开始减速
        * CoolDownState.ING      减速中 speed值会越来越小
        * CoolDownState.END      COOL DOWN模式结束
        *
        * speed:实时速度
        */
       @Override
       public void onCoolDownStatus(int status, int speed) {

       }
});

//无人运动监听
long detectionTime=45;//侦测无人运动时间，单位：秒

BoQunTreadmill.registerStateListener(
    new TreadmillStatusCallbacks.UnmannedStateDetectionCallback(detectionTime) {
       @Override
       public void onUnmannedStateDetection(boolean isUnmanned, long time) {
           //isUnmanned 当前是否是无人运动,当无人运动时间超过侦测时间就会变成true
           //time 开始正数无人运动时间
       }
});

//FTMS开启状态监听
BoQunTreadmill.registerStateListener(new TreadmillStatusCallbacks.FTMSStateCallback() {
            @Override
            public void onFTMSStateChange(boolean enable) {
                Log.d(TAG, "onFTMSStateChange: " + enable);
            }
        });
        BoQunTreadmill.setFTMSEnabled(true);

记要解除注册

BoQunTreadmill.unregisterStateListener(xxxx);

//或者,看个人应用场景和需求
BoQunTreadmill.unregisterAllStateListener();

OnTreadmillListener

函数 说明
void onResponseConfigInfo(TreadmillConfigInfo info) 回复下控配置信息
void onResponseSportStatus(@TreadmillSportState int state) 回复运动状态
void onResponseSportData(int heartRate, int speed) 回复运动资料
heartRate ：心率
speed：实时速
void onResponseSpeedLevel(int speed) 回复下控速度等级
void onResponseInclineLevel(int incline) 回复下控扬升等级
void onResponseFanLevel(int level) 回复风扇风力级别
void onExternalKeyEvent(@TreadmillKeyCodeint keyCode, int value) 面板外部键按下（UP,DOWN部分按钮可长按）触发回调
void onError(int errorCode) 发生错误回调ErrorCode,跑步机发生错误也会回调，不过下控通讯错误码还请厂商提供错误码列表

TreadmillSportState

参数 说明
TreadmillSportState.STOP 停止状态
TreadmillSportState.START 开始状态
TreadmillSportState.PAUSE 暂停状态
TreadmillSportState.COOL_DOWN 在COOL DONW模式
TreadmillSportState.ERROR 发生错误

BoQunTreadmillFactory

函数 说明
BoQunTreadmillFactory.init() 实例化跑步机工厂
BoQunTreadmillFactory.queryFactoryInfo() 查询跑步机工厂信息,回调onResponseFactoryInfo
BoQunTreadmillFactory.setFactoryInfo(int code, int value) 设置跑步机工厂信息
BoQunTreadmillFactory.startAutoCorrection() 开始自动校正,回调onResponseMotorAutoCorrection
BoQunTreadmillFactory.stopAutoCorrection() 停止自动校正,回调onResponseMotorAutoCorrection
BoQunTreadmillFactory.setOnTreadmillFactoryListener(OnTreadmillFactoryListener listener) 跑步机工厂监听器
BoQunTreadmillFactory.destroy() 销毁跑步机工厂实例

OnTreadmillFactoryListener

函数 说明
void onResponseFactoryInfo(TreadmillFactoryInfo info) 查询工厂信息回调
void onResponseMotorAutoCorrection(@AutoCorrectionState int status, int speed, int adc, int inclineVR) 自动校正回调

TreadmillConfigInfo

参数 类型 说明
getWattGroup() int 获取WATT 组别
getMinInclineLevel() int 获取最小扬升
getMaxInclineLevel() int 获取最大扬升
getMinSpeedLevel() int 获取最小速度
getMaxSpeedLevel() int 获取最大速度
getChipSpeedLevel() int 获取速度增量，加减速度时，应该用当前加或减getChipSpeedLevel()
hasIncline() boolean 是否有扬升
hasFan() boolean 是否有风扇

TreadmillKeyCode

参数 值 说明
TreadmillKeyCode.KEY_START_PAUSE 0x01 开始或暂停
TreadmillKeyCode.KEY_STOP 0x02 停止
TreadmillKeyCode.KEY_FAN 0x03 风扇
TreadmillKeyCode.KEY_SPEED_UP 0x04 速度加
TreadmillKeyCode.KEY_SPEED_DOWN 0x05 速度减
TreadmillKeyCode.KEY_INCLINE_UP 0x06 扬升加
TreadmillKeyCode.KEY_INCLINE_DOWN 0x07 扬升减
TreadmillKeyCode.KEY_COOL_DOWN 0x08 COOL_DOWN
TreadmillKeyCode.KEY_VOLUME_UP 0x09 音量加
TreadmillKeyCode.KEY_VOLUME_DOWN 0x10 音量减
TreadmillKeyCode.KEY_FAST_SPEED 0x1A 速度调整快速键
TreadmillKeyCode.KEY_FAST_INCLINE 0x1B 扬升调整快速键
TreadmillKeyCode.KEY_TURN_KNOB_FORWARD 0x1C 旋钮正转
TreadmillKeyCode.KEY_TURN_KNOB_REVERSE 0x1D 旋钮反转
TreadmillKeyCode.KEY_BACK 0x1E 返回键（可以自定义功能）
