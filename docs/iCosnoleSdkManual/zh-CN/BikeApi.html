脚踏车

脚踏车API

BoQunBike

参数 说明
BoQunBike.TYPE_SPORT_STATUS 运动状态，使用queryMachineInfo查询后回调onResponseSportStatus
BoQunBike.TYPE_LOAD_LEVEL 阻力等级，使用queryMachineInfo查询后回调onResponseLoadLevel
函数 说明
BoQunBike.init(OnBikeListener listener) 初始化脚踏车，返回一个布尔值，表示串行端口是否成功初始化。
BoQunBike.queryConfigInfo() 查询脚踏车配置信息阻力，扬升范围，状态，等等,回调onResponseConfigInfo，读取失败回调onError
BoQunBike.queryMachineInfo( int type) 查询脚踏车信息，参数type
BoQunBike.start() 开始运动,开始成功之后回调onResponseSportStatus,开始传输数据onResponseSportData
BoQunBike.pause() 暂停运动,暂停成功之后回调onResponseSportStatus，暂停状态期间不会传输运动数据
BoQunBike.stop() 停止运动,回调onResponseSportStatus
BoQunBike.setLoadLevel( int load ) 控制脚踏车阻力级别,回调onResponseLoadLevel，load 数值越大阻力越大，load 有范围限制，load 范围参数读取 BikeConfigInfo
BoQunBike.setInclineLevel( int incline ) 控制脚踏车扬升级别,回调onResponseInclineLevel，incline 数值越大倾斜度越高，incline 有范围限制，incline 范围参数读取 BikeConfigInfo
BoQunBike.setFanLevel( int level) 控制风扇风力级别,回调onResponseFanLevel，风力级别支持0-3段，机台没有风扇则不支持调整，获取机台是否有风扇状态BikeConfigInfo
BoQunBike.setWorkMode( int mode) 设置脚踏车下位机工作模式BikeWorkMode
BoQunBike.setFTMSEnabled( boolean enable) 启用或关闭FMTS，须搭配带含有FMTS功能的下控板使用，回调BikeStatusCallbacks.FTMSStateCallback
BoQunBike.setSerialPortEnabled( boolean enable) 启用和禁用串口，默认是启用状态,提供此方法是方便处理特殊情况，比如多个APP同时读取串口会造成互相干扰问题，使用此方法可避免
tips：并不会真正关闭串口，但绝对不会读取串口资料
BoQunBike.setSportData(long time, float speed, float distance, int calories, int rpm, int pulse, int watt) 当APP和电子表同步显示运动数据时，通常需要将运动数据发送到下位机。如果没有需求，请忽略

当APP和电子表同步显示运动数据时，通常需要将运动数据发送到下位机。如果没有需求，请忽略

- time 单位：秒 ，范围(0-65535)
- speed 单位：公里每小时，范围(0-65535)
- distance 单位：公里，范围(0-65535)
- calories 单位：卡，范围(0-65535)
- rpm 范围(0-65535)
- pulse 范围(0-65535)
- watt 范围(0-65535)
BoQunBike.setSportParams(@BikeSportParams int code, int value) 设置运动参数。如果没有需求，请忽略
有关参数详细信息，请参见 BikeSportParams
BoQunBike.setBikeListener(OnBikeListener listener) 脚踏车监听器（已弃用）
BoQunBike.registerStateListener(BikeStatusCallback callback) 注册状态回调监听器
BoQunBike.unregisterStateListener(BikeStatusCallback callback) 解除注册状态回调监听器
BoQunBike.unregisterAllStateListener() 解除所有注册状态回调监听器
BoQunBike.destroy() 销毁实例，释放资源

BikeWorkMode

参数 说明
BikeWorkMode.NORMAL TFT屏幕亮屏时应设置该状态,进入工作状态
BikeWorkMode.SLEEP TFT屏幕熄屏时应设置该状态,进入休眠模式
BikeWorkMode.SHUT_DOWN TFT关机时应设置该状态,释放资源

BikeStatusCallback

函数 说明
void onBikeSportStateChange(int status) 脚踏车状态变化回调

BikeStatusCallbacks

这个是类主要实现一下状态回调，目前支持监听FTMS状态,后续有需求再实现其它功能或者你可以继承BikeStatusCallback接口实现自定义状态回调

目前实现有FMTS开关状态回调

BoQunBike.registerStateListener(new BikeStatusCallbacks.FTMSStateCallback() {
            @Override
            public void onFTMSStateChange(boolean enable) {
                 //enable 启用和关闭状态
            }
        });

记要解除注册

BoQunBike.unregisterStateListener(xxxx);

//或者,看个人应用场景和需求
BoQunBike.unregisterAllStateListener();

OnBikeListener

函数 说明
void onResponseConfigInfo(BikeConfigInfo info) 回复下控配置信息
void onResponseSportStatus(@BikeSportState int status) 回复运动状态，参数：status
void onResponseSportData(int stepCount, int rpm, int heartRate) 回复运动资料
stepCount：步数（累加值）

rpm：每分钟转速

heartRate：手握心跳
void onResponseLoadLevel(int load) 回复下控阻力等级
void onResponseInclineLevel(int incline) 回复下控扬升等级
void onResponseFanLevel(int level) 回复风扇风力级别
void onExternalKeyEvent(@BikeKeyCode int keyCode) 面板外部键按下（UP,DOWN部分按钮可长按）触发回调，回复BikeKeyCode
void onError(int errorCode) 发生错误回调，参数见ErrorCode，跑步机下控错误码也会回调但是跑步机错误码要看对接马达厂商提供的错误码

BikeSportState

参数 说明
BikeSportState.STOP 停止状态
BikeSportState.START 开始状态
BikeSportState.PAUSE 暂停状态

BoQunBikeFactory

函数 说明
BoQunBikeFactory.init() 实例化脚踏车工厂
BoQunBikeFactory.queryLoadMotorStroke( int load) 查询指定阻力等级的阻力马达行程,回调onResponseLoadMotorStoke
BoQunBikeFactory.queryInclineMotorStroke( int incline) 查询指定扬升等级的扬升马达行程,回调onResponseInclineMotorStroke
BoQunBikeFactory.setWheelDiameter( int wheelDiameter) 设置机台轮径
BoQunBikeFactory.setLoadMotorStroke( int load, int adc) 设置指定阻力等级的阻力马达行程ADC值
BoQunBikeFactory.setInclineMotorStroke( int incline, int adc) 设置指定扬升等级的扬升马达行程ADC值
BoQunBikeFactory.startInclineAutoCorrection() 开始扬升马达自动校正，回调onResponseInclineAutoCorrection
BoQunBikeFactory.startInclineAutoCorrection(@AutoCorrectionDirection int direction) 开始扬升马达自动校正，回调onResponseInclineAutoCorrection

direction 校正方向

- AutoCorrectionDirection.AUTO 默认是自动方向
- AutoCorrectionDirection.UP
- AutoCorrectionDirection.DOWN
BoQunBikeFactory.stopInclineAutoCorrection() 停止扬升马达自动校正，回调onResponseInclineAutoCorrection
BoQunBikeFactory.setOnBikeFactoryListener( OnBikeFactoryListener listener) 脚踏车工厂模式监听器
BoQunBikeFactory.destroy() 销毁实例，释放资源

OnBikeFactoryListener

函数 说明
void onResponseLoadMotorStoke(int load, int adc) 回复指定阻力等级的马达行程ADC
void onResponseInclineMotorStroke(int incline, int adc) 回复指定扬升等级的马达行程ADC
void onResponseInclineAutoCorrection(@AutoCorrectionStateint state, int inclineADC) 回复自动校正的状态和ADC值，参数：state

AutoCorrectionState

inclineADC:值范围0-255，扬升马达校正中ADC会不停的变化，正常情况是从小到大，从大到小为一次完整校正

参数 说明
AutoCorrectionState.CORRECTION_START 扬升马达校正开始
AutoCorrectionState.CORRECTION_ING 扬升马达校正中
AutoCorrectionState.CORRECTION_STOP 扬升马达校正停止

BikeConfigInfo

参数 类型 说明
getClientId() int 获取客户ID，满足不同客户的定制化需求
getWattGroup() int 获取瓦特组别
getWheelDiameter() int 获取轮径
isManualControl() int 是否为手动控制
getMinLoadLevel() int 获取最小阻力等级
getMaxLoadLevel() int 获取最大阻力等级
getNegativeInclineLevel() int 获取负扬升等级
getMinInclineLevel() int 获取最小扬升等级
getMaxInclineLevel() int 获取最大扬升等级
hasIncline() boolean 机台是否配置有扬升
hasFan() boolean 机台是否配置风扇
hasFTMS() boolean 机台是否配置FMTS功能

BikeKeyCode

参数 值 说明
BikeKeyCode.KEY_START_PAUSE 0x01 开始或暂停
BikeKeyCode.KEY_STOP 0x02 停止
BikeKeyCode.KEY_LOAD_UP 0x03 阻力加
BikeKeyCode.KEY_LOAD_DOWN 0x04 阻力减
BikeKeyCode.KEY_INCLINE_UP 0x05 扬升加
BikeKeyCode.KEY_INCLINE_DOWN 0x06 扬升减
BikeKeyCode.KEY_FAN 0x07 风扇

BikeSportParams

参数 值 说明
BikeSportParams.SLOPE 1 倾斜度，用于FTMS中
