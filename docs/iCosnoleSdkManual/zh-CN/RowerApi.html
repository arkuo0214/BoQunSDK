划船机

划船机API

BoQunRower

参数 说明
BoQunRower.TYPE_SPORT_STATUS 运动状态，回调onResponseSportStatus
BoQunRower.TYPE_LOAD_LEVEL 阻力等级，回调onResponseLoadLevel
函数 说明
实例化划船机，返回一个布尔值，表示串行端口是否成功初始化。
BoQunRower.queryConfigInfo() 查询划船机配置信息阻力，风扇状态，等等,回调onResponseConfigInfo，读取失败回调onError
BoQunRower.queryMachineInfo(int type) 查询划船机信息
BoQunRower.start() 开始运动，回调onResponseSportStatus
BoQunRower.pause() 暂停运动，回调onResponseSportStatus
BoQunRower.stop() 停止运动，回调onResponseSportStatus
BoQunRower.setLoadLevel(int level) 控制划船机车阻力级别,回调onResponseLoadLevel，level数值越大阻力越大，level有范围限制，level范围参数读取 RowerConfigInfo
BoQunRower.setFanLevel(int level) 控制风扇风力级别,回调onResponseFanLevel，风力级别支持0-3段，机台没有风扇则不支持调整，获取机台是否有风扇状态RowerConfigInfo
BoQunRower.setWorkMode(@RowerWorkMode int mode) 设置划船机下位机工作模式
BoQunRower.setFTMSEnabled(boolean enable) 启用或关闭FMTS，须搭配带含有FMTS功能的下控板使用，回调RowerStatusCallbacks.FTMSStateCallback
BoQunRower.setSportData(long time, float speed, float distance, int calories, int spm, int pulse, int watt) 传递运动数据给下控
BoQunRower.registerStateListener(RowerStatusCallback callback) 注册运动状态监听器
BoQunRower.unregisterStateListener(RowerStatusCallback callback) 取消注册运动状态监听器
BoQunRower.unregisterAllStateListener() 取消注册所有运动状态监听器
BoQunRower.setOnRowerListener(OnRowerListener onRowerListener) 划船机监听器
BoQunRower.setSerialPortEnabled(boolean enable) 启用和禁用串口，默认是启用状态，提供此方法是方便处理特殊情况，比如多个APP同时读取串口会造成互相干扰问题，使用此方法可避免

tips：并不会真正关闭串口，但绝对不会读取串口资料
BoQunRower.destroy() 销毁实例，释放资源

RowerStatusCallback

函数 说明
void onRowerSportStateChange(int status) 运动状态变化

RowerStatusCallbacks

这个是类主要实现一下状态回调，目前支持监听FTMS状态,后续有需求再实现其它功能或者你可以继承RowerStatusCallback接口实现自定义状态回调

目前实现有FMTS开关状态回调

BoQunRower.registerStateListener(new RowerStatusCallbacks.FTMSStateCallback() {
            @Override
            public void onFTMSStateChange(boolean enable) {
                 //enable 启用和关闭状态
            }
        });

记要解除注册

BoQunRower.unregisterStateListener(xxxx);

//或者,看个人应用场景和需求
BoQunRower.unregisterAllStateListener();

</div>

RowerWorkMode

参数 说明
RowerWorkMode .NORMAL TFT屏幕亮屏时应设置该状态,进入工作状态
RowerWorkMode .SLEEP TFT屏幕熄屏时应设置该状态,进入休眠模式
RowerWorkMode .SHUT_DOWN TFT关机时应设置该状态,释放资源

OnRowerListener

函数 说明
void onResponseConfigInfo(RowerConfigInfo info) 回复下控配置信息
void onResponseSportStatus(@RowerSportState int status) 回复运动状态
void onResponseSportData(RowerSportData data) 回复运动资料
void onResponsePullAndPut(@IntRange(from = 0, to = 100) int progress, boolean isPull) 回复拉放状态及进度值
progress：拉放行程百分比

isPull = true 拉 ，isPull = false 放
void onResponseLoadLevel(int load) 回复阻力等级
void onResponseFanLevel(int level 回复风扇风力级别
void onExternalKeyEvent(@RowerKeyCode int keyCode) 面板外部键按下触发回调，（UP,DOWN部分按钮可长按）
void onError(int errorCode) 发生错误回调

RowerSportState

参数 说明
RowerSportState .STOP 停止状态
RowerSportState .START 开始状态
RowerSportState .PAUSE 暂停状态

BoQunRowerFactory

函数 说明
BoQunRowerFactory.init() 实例化划船机工厂
BoQunRowerFactory.queryFactoryInfo() 查询划船机工厂信息，回调onResponseFactoryInfo
BoQunRowerFactory.queryLoadMotorStroke(int load) 查询马达行程，回调onResponseLoadMotorStroke
BoQunRowerFactory.setLoadMotorStroke(int load, int adc) 设置马达行程
BoQunRowerFactory.setRepairDirection(@RowerRepairDirection int direction) 修复马达错误
direction：

- RowerRepairDirection.UP 修复上方
- RowerRepairDirection.DOWN 修复下方
BoQunRowerFactory.setFactoryInfo(RowerFactoryInfo info) 修改划船机工厂信息
BoQunRowerFactory.setOnRowerFactoryListener(OnRowerFactoryListener listener) 划船机工厂监听器
BoQunRowerFactory.destroy() 销毁工厂模式实例

OnRowerFactoryListener

函数 说明
void onResponseFactoryInfo(RowerFactoryInfo info) 回复工厂信息，参数RowerFactoryInfo
void onResponseLoadMotorStroke(int load, int adc) 回复马达行程，参数：load，adc
void onResponseMotorStatus(int status) 回复马达状态，参数：status

RowerMotorStatus

当发生错误时应使用 BoQunRowerFactory.setRepairDirection(@RowerRepairDirection int direction) 修复错误

参数 说明
RowerMotorStatus.NORMAL 正常
RowerMotorStatus.ERROR_UP 上方错误
RowerMotorStatus.ERROR_DOWN 下方错误

RowerSportData

参数 类型 说明
getTime() long 运动时间(单位：秒)
getSpm() int 每分钟桨频
getStroke() int 划桨次数
getWatt() int 功率（单位：w）
getDistance() int 运动距离，(单位：m）
getCalories() int 运动消耗卡路里（单位：kcal）
getTimePer500m() long 每500米多少时间(单位：秒)
getHeartRate() int 心率

RowerConfigInfo

参数 类型 说明
isLocalWattMode() boolean 本地WATT模式
hasFan() boolean 有风扇
isManualControl() boolean 手动控制
getMinLoadLevel() int 最小阻力
getMaxLoadLevel() int 最大阻力
getMinWatt() int 最小WATT
getMaxWatt() int 最大WATT

RowerFactoryInfo

参数 类型 说明
getDiameter() int 直径
getMagnet() int 磁石
getDistance() int 距离配比
getCalories() int 卡路里配比
getTimePer500m() int 每500米多少时间配比

RowerKeyCode

参数 值 说明
RowerKeyCode.KEY_START_PAUSE 0x01 开始或暂停
RowerKeyCode.KEY_STOP 0x02 停止
RowerKeyCode.KEY_LOAD_UP 0x03 阻力加
RowerKeyCode.KEY_LOAD_DOWN 0x04 阻力减
RowerKeyCode.KEY_FAN 0x05 风扇
